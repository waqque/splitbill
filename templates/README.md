# `templates/` — Шаблоны Jinja2

## Файлы

| Шаблон | Маршрут | Назначение |
|--------|---------|------------|
| `index.html` | `/` | Главная страница: список счетов, управление активным счётом, добавление данных |
| `bill.html` | `/bill/{id}` | Страница результатов: список долгов, сводка, навигация |

## Переменные контекста

### `index.html`

- `request` — объект запроса FastAPI
- `current_bill` — активный счёт (`Bill` или `None`)
- `bills` — список всех сохранённых счетов

### `bill.html`

- `request` — объект запроса
- `bill` — текущий счёт
- `debts` — список кортежей `(from_name, to_name, amount)`

## Адаптивность

- CSS-медиазапрос для экранов ≤768px
- Flexbox и Grid для форм
- Без JavaScript (чистый HTML/CSS)
