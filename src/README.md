# `src/` — Исходный код

## Модули

| Файл | Назначение | Ключевые функции |
|------|------------|------------------|
| `models.py` | Определение сущностей | `User`, `Item`, `Payment`, `Bill` |
| `storage.py` | Работа с файлами | `save()`, `load()`, `list_bills()` |
| `calculator.py` | Расчёт долгов | `compute_debts()`, `_simplify_debts()` |
| `web.py` | Веб-сервер | Эндпоинты `/`, `/add-user`, `/add-item` |

## Точка входа

```bash
python -m src.web
```
### Логирование
Ошибки выводятся в консоль. При production-запуске рекомендуется настроить файловое логирование.
